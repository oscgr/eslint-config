# @oscgr/eslint-config

Common opinionated eslint configuration from antfu to use for every web project

## How to maintain

Version bumping is required - feel free to check with `ncu` command. After bumping, copy commit hash in every project.
To install dependencies, use `pnpm`.

## Publish to NPM

```
npm publish
```
